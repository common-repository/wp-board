Add a message board in your WordPress! It's simple and easy to use: add,delete and modify messages in a second! 

== Description ==

English
A simple and easy to use plugin that adds a board useful to manage message among the users of the site.
You can add,delete,modify messages.
There is an admin menù and a menù in the admin-bar with the notification of new messages.

Italiano
Un plugin semplice e facile da usare che aggiunge una bacheca per lo scambio di messaggi tra gli utenti del sito
Aggiungi velocemente messaggi,modificali o cancellali.E' presente la notifica all'aggiunta di nuovi messaggi.
In più un menù per l'admin per poter modificare alcuni valori e un menù nell admin-bar per avere la notifica di nuovi messaggi mentre non sei nella bacheca

== Installation ==

It's easy to install your plugin:

1. Copy "wp-board.zip" in the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or you can install from your wordpress site using the "add plugin" feature.
